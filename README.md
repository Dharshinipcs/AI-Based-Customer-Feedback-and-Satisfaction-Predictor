# Customer Satisfaction Prediction System

An AI-based system that predicts customer satisfaction using structured feedback data from online and offline sources.

## Features
- Real-time satisfaction prediction
- QR-based and form-based feedback collection
- Reward points system
- Admin dashboard for insights
- Real- time Alerts to improve Business Value

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas, NumPy
- Plotly

## Model
- Random Forest Regressor
- Feature Engineering & Data Preprocessing

## Businesses can analyze feedback instantly and improve customer experience.
